### Getting started

- `git clone git@github.com:parkeragee/parker-boiler-app.git`
- `cd parker-boiler-app`
- `npm i` _(using Node >= v8.0.0)_

### Client commands

- Start app - `npm start`
- Deploy frontend app - `npm run deploy`

### API commands

- Start API - `npm run serve`
- Deploy API - `git push heroku master`
