### Client commands

- Start app - `npm start`
- Create optimized build - `npm run build`
- Deploy frontend app - `surge --domain app.teachercloset.com` (point to the build folder)

### API commands

- Start API - `npm run serve`
- Deploy API - `git push heroku master`
